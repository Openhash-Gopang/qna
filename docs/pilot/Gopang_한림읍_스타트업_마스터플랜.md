## 제1장. 한림읍 현황 분석

### 1-1. 기본 현황

| 항목 | 내용 |
|---|---|
| 행정 구역 | 제주특별자치도 제주시 한림읍 |
| 법정리 수 | 14개 리 (귀덕리·금능리·금악리·대림리·동명리·명월리·상대리·수원리·월령리·저지리·한림리·한수리·협재리·옹포리) |

| 위상 | 제주도 서부 지역의 중심 생활권 — 1956년 읍 승격 (제주시 다음으로 오래된 읍) |
| 특징 | 제주 서부 관광·어업·농업·축산의 복합 거점 |

### 1-2. 산업 구조

산업별 취업자 구성비를 보면 농업이 35%, 서비스업이 41%로 서비스업이 높은 비중을 차지하며, 예로부터 제주도 서부 지역의 중심지 역할을 담당해왔다.

| 산업 | 비중 | 주요 품목·내용 |
|---|---|---|
| 서비스업 | 41% | 관광·숙박·음식·소매·어업 서비스 |
| 농업 | 35% | 양배추·양파·맥주보리·감귤·선인장 |
| 축산업 | 기타 | 이시돌목장 중심 육우·면양·양돈 |
| 수산업 | 기타 | 한림항 — 제주 서부 연근해 수산 센터 |
| 제조·가공 | 기타 | 금능리 농공단지 (농축산물 가공, 제주맥주 양조장) |

**주요 관광 자원 (관광객 집중 지역)**
- 협재해수욕장 · 금능해수욕장 (제주 서부 최대 해변)
- 한림공원 (용암동굴·야자수·민속마을, 입장료 성인 12,000원)
- 비양도 (한림항 출발 도선 운항)
- 이시돌목장 · 금악리 목장 클러스터
- 월령리 선인장 군락지

## 제2장. 사업 목표 및 전략

### 2-1. 핵심 목표

```
Phase 1 (Month 1~6)
  자영업자 500명 + 시민 1,000명 = 실사용자 1,500명 확보

  제주 전체 60만 도민 확산 로드맵 수립

### 2-2. 고팡이 한림읍에서 해결하는 문제

**자영업자 500명 — 핵심 유인 메시지**

> "지금 카드 결제 수수료 2.5% 내고 계시죠?  
> 고팡 GDC 결제로 전환하면 0.2%로 줄어듭니다.  
> 재무제표는 자동 생성되고, 세금은 자동으로 납부됩니다."

| 현재 문제 | 고팡 해결책 | 담당 서비스 |
|---|---|---|
| 카드 수수료 2.5% | GDC 결제 수수료 0.2% | GDC |
| 재무제표 수동 작성 | 거래 즉시 자동 생성·갱신 | K-Tax + GDC |
| 세무사 비용 연 수백만 원 | AI 세무 자동화 | K-Tax |
| 물류·배송 수동 관리 | AI 자동 배분·추적 | K-Logistics |
| 홍보·온라인몰 부재 | AI가 판매 사이트 자동 생성 | K-Market |
| 법률 분쟁 시 변호사 비용 | K-Law 판결 예측 무료 | K-Law |

**시민 1,000명 — 핵심 유인 메시지**

> "등본 발급하러 읍사무소 가실 필요 없습니다.  
> 말 한 마디면 AI가 다 해드립니다."

| 현재 문제 | 고팡 해결책 | 담당 서비스 |
|---|---|---|
| 민원 처리 평일만 가능 | 24시간 365일 AI 대행 | K-Public |
| 응급 시 119 신고 번거로움 | AI가 자동 위치·상황 전송 | K-119 |
| 교육비 부담 | 166개 과목 AI 튜터 무료 | K-School |
| 이동·택시 불편 | 동선 겹침 최적 매칭 | K-Traffic |
| 보험료 과다 | AI 리스크 계산 기반 최적 보험 | K-Insurance |
| 법률 접근성 부족 | K-Law 판결 예측 무료 | K-Law |

## 제3장. 공공기관 협력 방안

### 3-1. 접촉 전략 — "비용 0원, 잃을 것 없음"

공공기관 협력의 핵심 원칙: **요구하지 않는다, 제공한다.**

모든 기관 접촉 시 공통 메시지:
> "도입 비용 없습니다. 기존 시스템 그대로 유지하시면 됩니다.  
> 저희가 무료로 설치하고 운영합니다. 3개월 써보시고 판단해 주세요."

### 3-2. 기관별 협력 시나리오

**① 한림읍사무소**

| 항목 | 내용 |
|---|---|
| 담당자 | 민원봉사팀장 |
| 제안 내용 | K-Public AI 민원 서비스 시범 운영 |
| 제공 가치 | 민원 처리 89% 시간 단축, 24시간 자동화, 직원 업무 부담 감소 |
| 요청 사항 | 읍사무소 내 QR코드 안내물 비치, 방문 민원인에게 고팡 안내 |
| 기대 효과 | "한림읍사무소 AI 민원 서비스" 공식 명칭 부여 → 도청 설득 레퍼런스 |

```

## 제4장. 사용자 온보딩 전략

### 4-1. 자영업자 500명 온보딩 시나리오

**1단계: 얼리어답터 30명 확보 (Month 1)**

대상: 한림읍 내 카드 결제 비중 높은 업종 우선
- 카페·음식점 (협재·금능 관광 상권)
- 숙박업 (펜션·게스트하우스)
- 편의점·마트
- 수산물 판매 (한림항 주변)

온보딩 방식:
```
① 직접 방문 → "5분만 보여드릴게요" 데모
② K-Tax로 지난달 카드 수수료 실제 계산 후 GDC 절감액 제시
③ 가입 즉시 재무제표 자동 생성 체험
④ 첫 GDC 결제 실습 (현장에서 바로)
```

**2단계: 입소문 확산 (Month 2~3)**

얼리어답터 30명의 실제 절감액을 수치로 공개 (동의 기반):
> "한림읍 ○○카페, 고팡 전환 후 월 카드 수수료 ○○만 원 절감"

지역 카카오톡 오픈채팅방, 맘카페, 자영업자 단체 채널에 공유.

**3단계: 500명 목표 달성 (Month 4~5)**

| 업종 | 목표 인원 | 핵심 유인 |
|---|---|---|
| 음식점·카페 | 150명 | 수수료 절감 + AI 재고 관리 |
| 숙박업 | 80명 | 예약·결제 자동화 |
| 소매·편의점 | 100명 | 재고 자동 발주 + K-Logistics |
| 농어업 | 100명 | 농수산물 물류 자동화 + 판매 사이트 |
| 기타 서비스 | 70명 | K-Tax 세무 자동화 |
