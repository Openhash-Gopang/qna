# Gopang × 제주도 파일럿 — 태스크포스 구성안
> AI City Inc. · 팀 주피터 · 2026년 6월  
> 근거: 14개 서브시스템 + 제주도 우선 적용 방안 v2.0

---

## 0. 구성 원칙

태스크포스(TF)는 다음 세 가지 원칙 아래 설계한다.

1. **최소 인원, 최대 커버리지** — 무자본 운영이므로 겸직·역할 통합이 기본. 전문성이 없는 역할은 외부 오픈소스 기여자·자문단으로 보완.
2. **서브시스템 담당제** — 14개 서브시스템 각각에 주담당(Owner)을 지정. 오너십 없는 서비스는 출시하지 않는다.
3. **제주 현장 인력 필수** — 원격만으로는 임계점을 달성할 수 없다. 제주 현지에 최소 1명 상주.

## 1. TF 전체 구조

```
총괄 리더 (TF장)
├── A. 기술 팀 (Tech)
│   ├── 풀스택 개발자 (서브시스템 유지·확장)
│   ├── AI/프롬프트 엔지니어 (14개 서브시스템 AI 품질)
│   └── 인프라 엔지니어 (Cloudflare·Supabase·OpenHash)
├── B. 사업·파트너십 팀 (Biz)
│   ├── 제주 현장 담당자 (상주)
│   ├── 기관 협력 담당 (소방청·경찰청·도청·한국은행)
│   └── 자영업자·소상공인 온보딩 담당
├── C. 운영·데이터 팀 (Ops)
│   ├── KPI 모니터링 담당
│   ├── 사용자 피드백·인터뷰 담당
│   └── 법무·컴플라이언스 담당 (GDC 샌드박스, 전자정부법)
└── D. 외부 자문단 (Advisory)
    ├── 의료 자문 (K-Health·K-119)
    ├── 법률 자문 (K-Law·K-Police·GDC 샌드박스)
    ├── 금융 자문 (GDC·K-Stock·K-Insurance)
    └── 행정 자문 (K-Public·K-Democracy)
```

## 2. 핵심 포지션별 역할 정의

### TF장 (총괄 리더)
**필요 역량:** 기술 이해 + 기관 협상 + 제주 네트워크  
**주요 업무:**
- 제주도청·소방청·경찰청·한국은행 고위급 MOU 협상
- 주간 화상회의(매주 수요일 11:00) 주재
- R&D 과제·샌드박스 신청 최종 서명
- 언론·학회 대외 발표

**이상적인 배경:** 전직 공무원(행정) 또는 스타트업 대표 경험자. 제주도 연고 있으면 플러스.

---

### A-1. 풀스택 개발자 (2명 권장, 최소 1명)
**필요 역량:** JavaScript(Node.js/Cloudflare Worker) + Supabase + HTML/CSS  
**주요 업무:**

| 담당 서브시스템 | 현재 상태 | 즉시 필요 작업 |
|---|---|---|
| K-119 · K-Police | 운용 중 | SMS 폴백, 오프라인 대응 로직 |
| K-Insurance | 운용 중 | `pdv_consent_requests` 테이블 완성, 파이프라인 테스트 |
| K-Market | 운용 중 | GPID 대량 등록 툴, 제주 상품 온보딩 |
| K-Public | 개발 완료 | 제주시청 챗봇 통합 API |
| K-Traffic | 파일럿 준비 | 렌터카 QR 인증 모듈 |
| GDC | 운용 중 | 환전 봇, POOL 잔고 모니터링 |
| 공통 | — | `hondi.net/jeju-dashboard` 공개 대시보드 구축 |

**채용 전략:** GitHub `Openhash-Gopang` 오픈소스 기여자 중 선발 우선. 인턴·부트캠프 수료생 활용 가능(무급 기여 → 성과 후 유급 전환).

---

### A-2. AI/프롬프트 엔지니어 (1명)
**필요 역량:** LLM 프롬프트 설계 + DeepSeek/Claude/Gemini API + JSON 파싱  
**주요 업무:**

| 서브시스템 | AI 과제 |
|---|---|
| K-Law | v15.1 판결 정확도 향상·사건 유형 추가 |
| K-Health | 응급 키워드 탐지 고도화, 편향 감사 |
| K-School | 26개 언어 커리큘럼 품질 QA |
| K-Public | 웹 검색 기반 법령 실시간 조회 정확도 |
| K-Stock | 89개 자산군 분류 모델 유지 |
| K-Democracy | 안건 JSON 합리성 점수 검증 |
| 공통 | SP(시스템 프롬프트) 버전 관리·문서화 |

**채용 전략:** KAIST·고려대 AI 연구실 석사생 협업 (공동연구 논문 공저 제안).

---

### A-3. 인프라 엔지니어 (1명, 겸직 가능)
**필요 역량:** Cloudflare Workers + Supabase + OpenHash + 보안  
**주요 업무:**
- `gopang-proxy.tensor-city.workers.dev` Worker v4.x 유지·업그레이드
- Supabase 테이블 스키마 관리 (14개 서브시스템 DB 통합)
- OpenHash PDV 분산원장 노드 제주 로컬 노드 설치
- 무료 플랜 한도 모니터링 → 트래픽 급증 시 캐시 최적화 대응

---

### B-1. 제주 현장 담당자 (1명, 제주 상주 필수)
**필요 역량:** 제주 지역 네트워크 + 영업·설득력 + 기본 IT 이해  
**주요 업무:**
- 로컬 앰배서더 20곳 모집·관리 ("고팡 파트너 가게" 현판 설치)
- 공항·버스터미널·관광안내소 키오스크 유지 보수
- 자영업자 GDC 입금 유도·K-Insurance 가입 설명회 현장 진행
- 매주 금요일 시민 인터뷰 10명 (음성 녹음 → AI 요약 전송)
- 렌터카 업체(롯데·AJ·제주패스) 실무 담당자 접촉

**이상적인 배경:** 제주 출신 또는 거주 중인 20~40대. 카카오·배민 지역 영업 경험자 우대.  
**보상:** 초기 무급 → GDC 인센티브(월 성과에 따라 500~2,000 GDC) → 유급 전환.

---

**필요 역량:** 공문 작성 + 기관 프로토콜 이해 + 프레젠테이션  
**주요 업무:**

| 대상 기관 | 목표 | 타임라인 |

| 제주도청 스마트시티과 | 간담회 → 스마트시티 예산 연동 과제 편성 | Week 1~2 |
| 제주도 소방청 | K-119 공식 보조 채널 MOU | Month 1 |
| 제주지방경찰청 | K-Police 시범 운용 계약 | Month 1 |
| 한국은행 제주 지점 | GDC CBDC 공동연구 협정 | Month 2 |
| 제주관광공사 | 관광객 앱 설치 캠페인 협업 | Month 1 |
| 제주세무서 | K-Tax 세무행정 연동 | Month 3 |
| 제주 농협·수협 | K-Logistics 감귤·전복 물류 파일럿 | Month 3 |

---

### B-3. 자영업자·소상공인 온보딩 담당 (1명, B-1 겸직 가능)
**필요 역량:** 영업 + SNS 마케팅 + 지역 커뮤니티 활동  
**주요 업무:**
- GDC 입금·K-Insurance 가입 설명회 기획·진행
- 제주MBC·제주일보·제주의소리 기고문 작성 (AI 초안 → 담당자 검토)
- 인스타그램·카카오 채널 운영 ("고팡 파트너 가게" 콘텐츠)
- 지역화폐(탐나는전) 운영사와 GDC 연계 협의

---

### C-1. KPI 모니터링 담당 (1명, A-3 겸직 가능)
**주요 업무:**

**주요 업무:**
