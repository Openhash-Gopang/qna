### 1-1. K-Law — 판결 시뮬레이터
**URL:** `klaw.hondi.net`

사건의 핵심 사실관계 입력 → AI가 대법원 판결 예측 (K-Law v15.1 방법론 적용)

**처리 단계**
```
STEP 0: 사건 분석·명제 설정
STEP A: 판례 법리 분석
STEP B: 법률관계 확정·판결문 생성
STEP C: 확신도 산출·자기검증
```

**기술 스펙**

| 항목 | 내용 |
|---|---|
| AI 엔진 | DeepSeek V4 Pro (무료 내장) / Claude Sonnet 4 / Gemini 2.5 Pro (BYOK) |
| 사건 분류 | 민사·형사·행정·가사·소액·노동 (TYPE 01~06) |
| 출력 지표 | 법리·사실·종합 확신도(X/10), 복잡도(X/10), 대법원 일치도(%) |
| 데이터 저장 | Supabase `klaw_cases` + `klaw_sessions` |
| 인증 | 고팡 SSO → `user.ipv6` PDV 자동 보고 |

**수익화 포인트**
- DeepSeek 무료 플랜 운용 중 → Claude·Gemini는 BYOK(사용자 직접 키 입력)
- 누적 판결 데이터 → 법무법인 B2B 세일즈 근거
- "허위 판례 인용 금지" 가드레일 → 신뢰도 기반 프리미엄 전환 근거

### 1-2. K-119 — AI 응급출동 서비스
**URL:** `911.gopang.net` | **현황:** 제주도 시범 서비스

**핵심 지표**

| 지표 | 수치 |
|---|---|
| 평균 현장 도착 | 5.8분 |
| AI 대기 | 24시간 상시 |
| 중증도 분류 | 4단계 (1~4급) |

**3계층 아키텍처**

| 계층 | 구성 | 역할 |
|---|---|---|
| Layer 1 — 시민 인터페이스 | K-119 앱 | 24시간 AI 채팅, GPS, 중증도 분류, 응급처치 가이드 |
| Layer 2 — AI 지휘 엔진 | SeverityEngine · DispatchEngine · HospitalEngine | 중증도 분류, Haversine 최적 배치, 자동 출동 |
| Layer 3 — 종합상황실 | 대시보드 | 실시간 출동 지도, 6뷰 통합, 출동 승인·오버라이드 |

**4단계 중증도 → 자동 대응**
```
1급(위독) → 즉시 자동 출동 · Haversine 최근접 소방서·구급대 선발
2급(중증) → 출동번호 발급 후 배치
3급(중등) → AI 대기 + 상황 모니터링
4급(경증) → 응급처치 가이드 · 보건소 연결 권고
```

**수익화 포인트**
- 소방청 공식 연동 → 국내 최초 AI 119 접수 → 행안부·소방청 B2G R&D
- 웨어러블 연동 예방적 감지 → 제주도 스마트시티 예산 연계
- NDMS(국가재난안전시스템) 연동 → 재난 빅데이터 연구기관 협력
- PDV 출동 기록 → 보험사 사고 처리 자동화 (K-Insurance 시너지)

| 항목 | 우버·카카오택시 | K-Traffic |

### 1-6. K-Health — AI 평행 병원
**URL:** `health.gopang.net`

**핵심 지표**

| 지표 | 수치 |
|---|---|
| 연결 의료 기관 | 전국 87,000개 |
| 의학 DB | PubMed 3,700만 편 + WHO + ICD-11 |
| 진단 정확도 향상 | 87% (파일럿 기준) |
| 평균 예약 완료 | 3분 |
| 암호화 방식 | MedCrypt 3단계 (AES-256-GCM → Paillier → RSA-OAEP) |

**7단계 AI 의료 워크플로**
```
1. 증상 파악   → AI 대화·사진 분석, 전 생애 PDV 참조
2. 확률적 분석 → ICD-11 기반 질환 확률 순위 (예: 위장염 55%, 식중독 25%)
3. 기관 추천   → 경증→보건소, 중등→의원, 중증→종합병원, 응급→응급실+앰뷸런스
4. AI 예약     → K-Health AI가 기관 AI와 직접 협상 → 평균 3분 완료
5. 사전 브리핑 → 증상·PDV 이력 의사에게 자동 전달 → 진단 정확도 87% 향상
6. 처방→약국  → 인근 약국 자동 전송 → 환자 도착 전 약 준비
7. 예후 추적   → 1일·3일·1주 후 자동 확인 → 호전 없으면 상위 기관 전원 제안
```

**수익화 포인트**
- 보건소 1순위 원칙 → 건강보험공단·복지부 B2G 협력
- K-119 + K-Health → **응급 의료 풀스택**
- K-Insurance + K-Health → AI 건강 데이터 기반 개인화 보험료 산출
