## Personal Agent ↔ Institutional Agent 상호작용 기반 전자소송·범용 서비스 자동화 특허
- 발명명: Personal Agent와 Institutional Agent 간 상호작용을 통한 전자소송 및 범용 서비스 자동화 시스템
- Personal Agent: 개인에게 할당, 개인이익 복무. PDV(개인정보금고) 연동.
- Institutional Agent: 법원·병원·학교·시장 등 기관 대변, 공적 서비스 제공.
- 핵심: Personal Agent가 대법원 전자소송 Institutional Agent와 직접 소통 → 법무 자동화
- 동일 Agent 상호작용 모델을 의료·교육·행정·상거래로 확장 가능한 **범용 Agent 생태계**
- 기존 전자소송 문제 해결: 법률 전문지식 부족 / 개인정보 전체 노출 / 복잡한 절차
- K-Law, K-Health, K-School, K-Public과 직접 연결되는 아키텍처 특허

## 특허 요약: Personal Agent & Institutional Agent 기반 서비스 자동화
- 이중 Agent 구조: Personal Agent(개인 대변) ↔ Institutional Agent(기관 대변)
- **zk-SNARK** 프로토콜: 필요 정보만 선별적 추출하는 영지식증명 개인정보 보호
- 적응형 재시도 메커니즘으로 서비스 안정성 보장
- 적용 도메인: 법률·의료·교육·행정·상거래 (범용 확장 플랫폼)
- 실증: 2,934건 / 상호작용 성공률 51% → 100% (적응형 재시도 메커니즘)
- 기술 가치: 3,000억~6,000억 원 / IRR 30% 이상
