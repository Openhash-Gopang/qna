# K-119 (911.hondi.net)
- 평균 현장 도착: **5.8분**
- 중증도 4단계: 1급(위독)→즉시 자동 출동 / 2급(중증) / 3급(경증) / 4급→가이드 제공
- 출동번호 형식: FD-YYYYMMDD-NNNN
- 대응 재난 8종: 화재·구조·구급·교통·수난·화학/특수·자연재해·기타
- 매칭: Haversine 알고리즘으로 최근접 소방서·구급대 자동 선발
- **AI 독단 없음**: 사람 소방관이 출동 승인·오버라이드 권한 보유
- 오출동 방지: 1급 자동 출동 전 AI 음성 확인 2회
- 오프라인 대응: SMS 폴백 + 위치 좌표 로컬 저장 후 재전송
- 다국어: 영어·중국어·일본어 UI 제공
